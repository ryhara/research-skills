# Survey Skill — CV 中心の論文サーベイ

## 目的
ユーザーが指定するトピック (例: "3D Gaussian Splatting", "diffusion-based video generation") について、主要な Computer Vision / ML 系会議・ジャーナルから関連論文を収集し、テーブル形式の一覧 + 各論文の簡潔な解説を提示する。

## 検索条件 (デフォルト)

### 対象年度
- **2018 年〜現在** (ユーザー指示が優先)

### 対象会議・ジャーナル
Computer Vision を重視しつつ、以下を対象とする:

- **CV 主要会議**: CVPR, ICCV, ECCV, WACV, BMVC, ACCV, 3DV
- **CV 系ジャーナル**: TPAMI, IJCV, IEEE TIP (ITIP), Pattern Recognition
- **CV 系ワークショップ**: CVPRW, ECCVW, ICCVW
- **パターン認識・画像処理**: ICPR, ICIP
- **機械学習主要会議**: ICLR, ICML, NeurIPS, AAAI
- **ロボティクス**: IROS
- **グラフィクス**: SIGGRAPH, SIGGRAPH Asia, ACM TOG, TVCG
- **プレプリント**: arXiv
- **オープンアクセス**: IEEE Access

### 優先度
- **最優先**: ユーザーが明示した年度・会議の条件
- **次点**: Computer Vision 分野での引用数・話題性・新規性
- 学会名や年度の指定がない場合は上記デフォルトを適用

## 実行手順

1. **クエリ分析**
   - ユーザー指示からトピック・キーワードを抽出
   - 年度・会議の上書き指定があれば優先して適用
   - 曖昧な場合は検索を進めつつ、必要に応じて確認を挟む

2. **論文収集** (WebSearch / WebFetch を活用)
   - arXiv (https://arxiv.org) をキーワード検索
   - Google Scholar 風の検索クエリ (`"keyword" site:openaccess.thecvf.com` 等) で CVF 公開論文を探す
   - Semantic Scholar / Papers with Code も補助的に利用
   - 並列化できる検索は Agent (subagent_type=general-purpose) で並行実行

3. **フィルタリング**
   - 対象会議・ジャーナルに該当するかを確認
   - 2018 年以降かを確認
   - CV との関連度が低いものは除外、または別枠で注記

4. **テーブル出力** (必ずこの順)

   ```markdown
   | 会議/ジャーナル | 年 | タイトル | URL | 要約 (1-2 行) |
   | --- | --- | --- | --- | --- |
   | CVPR | 2023 | ... | https://... | ... |
   ```

   - 会議名は略称で統一 (CVPR, ICCV, NeurIPS など)
   - URL は arXiv / CVF Open Access / 公式ページのいずれか
   - 要約は 1〜2 行で「何をした論文か」を簡潔に

5. **各論文の解説**
   テーブルの後に、各論文について以下の形式で解説:

   ```markdown
   ### [N] {タイトル} ({会議} {年})
   - **課題**: 何を解こうとしているか
   - **手法**: コアアイデア (2-3 行)
   - **結果**: 主要な貢献・性能
   - **CV 分野での位置づけ**: なぜ重要か / 何が新しいか
   ```

6. **最後にまとめ**
   - トレンド・系譜 (例: "2020 年の X から 2023 年の Y への発展")
   - 読む優先順位の提案 (例: "まずこの 3 本を読むと俯瞰できる")

## 注意事項

- **推測で存在しない論文を作らない**: 検索結果に裏付けのない論文は絶対に載せない
- **URL は実在確認**: arXiv ID / CVF ページなど、アクセス可能な URL のみ掲載
- **会議名の判別**: arXiv プレプリントで会議採択情報がない場合は「arXiv (未採録 or 不明)」と明記
- **件数**: 特に指示がなければ 5〜10 本程度。多すぎると解説が薄くなる
- **ユーザー条件の優先**: 「CVPR のみ」「2023 年以降」などの指示があれば、デフォルト条件より優先する
- **検索クエリの工夫**: 単一キーワードで不十分な場合は類義語・関連概念でも検索する (例: "NeRF" なら "neural radiance field" も)

## 出力例 (抜粋)

```markdown
# "monocular depth estimation" サーベイ (2018-現在, CV 重視)

## 論文一覧

| 会議 | 年 | タイトル | URL | 要約 |
| --- | --- | --- | --- | --- |
| CVPR | 2024 | Depth Anything | https://arxiv.org/abs/... | 大規模ラベルなしデータで...  |
| ICCV | 2023 | ZoeDepth | https://arxiv.org/abs/... | 相対深度と絶対深度を統合... |
| ...

## 各論文の解説

### [1] Depth Anything (CVPR 2024)
- **課題**: 汎化性能の高い単眼深度推定
- **手法**: 6200 万枚のラベルなし画像を Teacher-Student で活用
- **結果**: ゼロショットで既存手法を凌駕
- **位置づけ**: MiDaS 系譜の大規模化版。データ中心のアプローチを示した
```
